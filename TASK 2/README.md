# HKMU Task 2: Priority Print Queue (Max Heap & Heap Sort)
**Student**: Dong QiZhen (13937844)

## Overview
Implemented a **Max Heap** from scratch to manage a campus print queue. 
High-priority tasks (e.g., Exam Drafts) are always processed before low-priority ones.

## Key Logic
- **Heapify Up**: Used when inserting new jobs O(log n).
- **Heapify Down**: Used when extracting the top priority job O(log n).

## How to Run
1. Navigate to `TASK 2` folder.
2. Run: `python heap_sort.py`
